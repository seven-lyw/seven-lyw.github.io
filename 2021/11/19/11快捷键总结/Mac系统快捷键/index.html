Mac系统快捷键

Seven 2021-11-19 10:59:41
Categories： 11快捷键总结 Tags： 快捷键

Mac中符号的含义

符号 苹果的键盘 普通的键盘
⌘ command window
⌥ option alt
⇧ shift shift
⇪ caps lock caps lock
⌃ control control
↩ return return
⌅ enter enter

常用快捷键

  • 多屏幕展示 系统默认按control+向上键
  • 应用程序窗口 系统默认按control+向上键
  • 切换屏幕 系统默认按control+左／右
  • 显示桌面 F11
  • 快速当前应用切换 command+tab
  • 输入法切换快捷键 control + space (默认)
  • 系统偏好设置 > keyboard中设置
  • 打开关闭Dock的隐藏 option + command + D (默认)
  • 浏览器切换control + table
  • ⌃⌘F 切换全屏模式
  • 隐藏文件：command+shift+.
